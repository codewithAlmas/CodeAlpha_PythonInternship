# Hangman Game

A simple text-based Hangman Game developed using Python as part of the CodeAlpha Python Programming Internship.

## 📌 Features
- Random word selection
- Letter guessing system
- Limited number of tries
- Win/Lose conditions
- Console-based gameplay

## 🛠 Technologies Used

- Random Module
